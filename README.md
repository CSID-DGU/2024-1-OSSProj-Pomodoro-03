### 프로젝트 내용  
**[ 프로젝트 개요 ]**
- 배경 : 자취생들이 생활정보 및 배달/공동구매 쉐어를 진행할 수 있는 커뮤니티의 필요성
- 목적 : 혼자살때 불편했던 여러 부분을 커뮤니티에서 해결할 수 있도록 하고, 공동구매와 배달쉐어를 통한 생활비 절약을 돕는다.
